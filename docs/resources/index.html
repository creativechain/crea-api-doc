Crea Developer Portal - Resources

Overview

Crea has an active developer community that is constantly innovating on the blockchain. While the their presence on this page doesn’t constitute endorsement, it’s likely a few of these projects could be beneficial for your Crea idea.

Client Libraries

Radiator - https://github.com/inertia186/radiator

Radiator is a Ruby API client to interact with the crea blockchain.

Crea-JS - https://github.com/creativechain/crea-js

Pure JavaScript Crea crypto library for node.js and browsers. Can be used to construct, sign and broadcast transactions in JavaScript.

DCrea - https://github.com/jnordberg/dcrea

A Typescript Crea crypto library for node.js and browsers. Creary Inc also maintains a version of this library

Beem - https://github.com/holgern/beem

A python library to interact with the CREA blockchain. It includes the CLI tool beempy.

PHP Crea - https://github.com/lukestokes/php-crea-tools

Various tools and scripts written in PHP for exploring the CREA blockchain.

CreaJ - https://github.com/marvin-we/crea-java-api-wrapper

An API Wrapper for Crea written in Java

GoCrea - https://github.com/go-crea/rpc

Golang RPC client library for Crea

CreaClientRS - https://github.com/cyberpunk-ventures/crea-client-rs

Client library for Crea blockchain built with Rust

Tools

ChainSync - https://github.com/aaroncox/chainsync

A simple library to stream blocks and operations for digesting into other mediums.

CreaIAPI - http://crea.ecrea.ws/

Interactive Crea API swagger, open-source script allows you to interact with blockchain.

CreaSQL - http://www.creasql.com/

A private Microsoft SQL server database with Crea blockchain data, subscription based, allows you to do flexible queries and analyze blockchain data.

eSync - https://github.com/eCreaApp/esync

eSync extracts Crea blockchain data and saves into Mongodb, written in Nodejs.

Many more projects and tools can be found at https://creaprojects.com

CreaConnect Libs

Most people who will be implementing OAuth2 will want to find and utilize a library in the language of their choice. These libraries, while not necessarily built by Creary, should support the CreaConnect API.

If you’re wondering what CreaConnect is, go here.

CreaConnect SDK - https://github.com/creativechain/creaconnect-sdk

An official javascript library for utilizing CreaConnect.

crea-connect-firebase-function - https://jakipatryk.github.io/creaconnect-firebase-functions/

A TypeScript library that can help you build applications with CreaConnect and Firebase.

Sc2 SDK PHP - https://github.com/hernandev/sc2-sdk-php

Easily integrate CREA blockchain into your PHP applications, though CreaConnect.

Social Auth CreaConnect - https://pypi.python.org/pypi/social-auth-creaconnect/0.0.2

Pluggable authentication backend for python-social-auth, that allows authentication via CreaConnect (v2).

creaconnect-python-client - https://github.com/emre/creaconnect-python-client

creaconnect-python-client is a simple yet powerful library to interact with the Creaconnect.

omniauth-creaconnect - https://rubygems.org/gems/omniauth-creaconnect/versions/0.1.0

Ruby Omniauth2 Strategy for CreaConnect.

SwiftyConnect - https://github.com/caspernikus/SwiftyConnect

CreaConnect Library for iOS / Swift.

CreaConnect4j - https://github.com/hapramp/creaconnect4j

Creaconnect SDK for Java.

Community & Help

Developer Advocate

The members of the Creary Inc. development team are currently the main contributors to the Crea blockchain software. They oversee the open source Crea GitHub repository, and maintain many of the open source libraries that developers use.

Creary’s Developer Advocate is here to make the process of developing for the Crea platform as smooth as possible. If you have any comments, concerns, or suggestions about how to improve the experience of developing applications with the Crea blockchain - please reach out.

The D.A. receives a lot of emails, but will respond as quickly as possible.

CreaDevs Chat - https://discord.gg/B29Bbng

CreaDevs chat is a public Discord chat community where members of the Crea development community go to discuss Crea development, and other related topics. It is a great place to go to ask questions, meet other developers that are working on Crea projects, share tips and code snippets, and discuss the items you are working on.
